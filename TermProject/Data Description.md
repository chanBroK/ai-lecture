# File
- train.csv - 학습을 위한 데이터 [2005,2006,2007년 데이터]
- test.csv - 테스트를 위한 데이터 [2008년 데이터]
- submission.csv - 제출 양식 데이터

# Columns
- area_name - 행정동 이름
- min_temperature - 관측된 최저 기온
- yaer - 연도
- gas_usage - 가스 사용량
